Detailed Description

Classe pour gérer le rendu graphique 2D des objets projetés depuis une scène 3D.

Cette classe utilise SDL pour dessiner les objets projetés en 2D à partir d'une scène 3D. Elle fournit des méthodes pour configurer l'écran, dessiner des formes et afficher le rendu.
